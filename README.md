# Next.js Redux, Saga, Persist, Logger Starter

## 🚀 Getting Started

Run the following command to create a new project with this Starter:

```bash
yarn create next-app <app_name> -e https://github.com/ShyamPanchal/next-rspl-starter

# or

npx create-next-app <app_name> -e https://github.com/ShyamPanchal/next-rspl-starter
```

Once the project and dependencies are finished installing, you can navigate to that directory and start up the development server with:

```bash
yarn dev

# or

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see your new project!

## 🧐 What's inside?

This Starter includes

- [React-Redux](https://react-redux.js.org/)
- [Redux-Saga](https://redux-saga.js.org/)
- [Redux-Persist](https://www.npmjs.com/package/redux-persist)
- [Redux-Logger](https://www.npmjs.com/package/redux-logger)

The rest of the Starter is based off of the Next.js default starter.
